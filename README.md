## Installation Guide

Easily deploy your Casino Platform on Ubuntu 20.04/22 or Debian 10/11 with these steps.


### Manual Installation Steps

1. **Edit Configuration Files**:  
   - `sql.php`, `config/config.php`, `template/js/app.js` (Line: 471), `server/config.js`
2. **Install Node.js** (latest LTS version)
3. **Server Setup**:  
   - Navigate to `server/` and run `npm install`
4. **Import Slot Games**:  
   - Execute `node server/addGames.js`
5. **Install PM2 for Process Management**:  
   - `npm install -g pm2`
6. **Start the Server**:  
   - `pm2 start server/server.js`
